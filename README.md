这个项目是官网 landing page， 这个页面只有当真正的官网在维护或者更新时才使用。开源project

目前已完成✅


## Setup

Make sure to install dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```
